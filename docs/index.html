Contributing

Make a pull request (detailed instructions on how to make one are in this document). Typically it should just be necessary to modify or add to only the .tm sources in the src/directory–-and modify accordingly index.tm. The html files are then generated by one of the maintainers after the pull request is merged (hopefully this will be automated in the near future). Once the changes are pulled in they will become immediately publicly visible on the website.
